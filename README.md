# ①課題名
API追加）世界の小さな日常 - 子供たちの世界への好奇心を育てるグローブダッシュボード

## ②課題内容（どんな作品か）
- 子供たちが世界の国々について楽しく学べるインタラクティブな3Dグローブダッシュボード
- 会話形式の学習システムで、人口、言語、宗教、気候、民族、朝食文化などを楽しく学べる（今回はベトナムを例として表示：今後DBを揃えて各国対応）
- ユーザー管理システムと管理者機能を備えた完全なWebアプリケーション
